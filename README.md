Sometimes I build little Factorio tools for myself. Figured, I'll make a nice website for them: https://arnoldsmith86.github.io/factorio-tools/

Consider all the code public domain (CC0). Most of it was written by AI anyway. Except for the files in `data/`, which are from Factoriolab (MIT) and Factorio.

---

Build it locally with `build.sh` which just replaces a bunch of `@INCLUDE` strings with the contents of the files they point to. Watch for changes with `./auto-build.sh`.
